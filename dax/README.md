# DAX

Propósito: biblioteca de medidas DAX, padrões e snippets para Power BI (medidas, KPIs, time intelligence, formatação).

Organização:
- `dax/measures/`: medidas prontas (uma por arquivo ou agrupadas por assunto).

Sugestão de padrão:
- Use `.dax` ou `.md` (quando quiser explicar contexto/uso).
- Nomes no formato: `assunto__nome_da_medida.dax` (ex.: `vendas__receita_liquida.dax`).
- Documente dependências (tabelas/colunas) e premissas.
