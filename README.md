# bi-ai-toolkit

Toolkit para acelerar entregas de BI com IA: biblioteca de prompts, queries SQL, medidas DAX, templates de dashboard e documentação.

## Estrutura
- `prompts/`: prompts reutilizáveis (biblioteca e snippets)
- `sql/`: queries SQL organizadas por assunto/camada
- `dax/`: medidas DAX e padrões de modelagem
- `templates/`: templates e assets de dashboards
- `docs/`: documentação (guias e referências)

## Skill (Copilot)
- `.github/skills/powerbi-advanced-pattern/`: skill para gerar um kit reutilizável (SQL + DAX + insights) para Power BI.

> Dica: mantenha nomes descritivos e consistentes para facilitar busca e reuso.
